Consejos basicos para utilizar Atributos

Son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios.

Uno de los atributos son las etiquetas < a > que son las que nos definen a un hipervinculo, el cual es el que nos va a direccionar a la pagina de enlace

El comando utilizado seria el siguiente

< a href="https://www.w3schools.com">Visit W3Schools

Visit W3Schools

Otro de las etiquetas a utilzar es < img > el sual se utiliza para agregar imagenes a la pagina HTML, y en este casio el atributo a utilizar seria SRC que sera el que muestre la ruta a la imagen que se mostrara

El comando utilizado seria el siguiente

< img src="img_girl.jpg">

Atributos de ancho y alto, también debe contener los atributos de ancho y alto, especifican el ancho y el alto de la imagen en píxeles

El comando utilizado seria el sigiente

< img src="img_girl.jpg" width="500" height="600">

Consejos basicos para utilizar Párrafos HTML

Un párrafo siempre comienza en una nueva línea y suele ser un bloque de texto.

El elemento HTML

define un párrafo.Un párrafo siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco (un margen) antes y después de un párrafo.

El comando utilizado seria el siguiente

< p>This is a paragraph.

< p>This is another paragraph.

Reglas horizontales HTML La etiqueta < hr> define una ruptura temática en una página HTML y se muestra con mayor frecuencia como una regla horizontal.

El comando utilizado seria el siguiente

< h1>This is heading 1

< p>This is some text.

< hr> < h2>This is heading 2 < p>This is some other text.
